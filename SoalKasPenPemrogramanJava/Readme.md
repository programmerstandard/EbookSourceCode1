## Cara Compile Java

### Lewat Command Line

Anda akan mengetikkan perintah di aplikasi yang disebut Command Prompt.

* Dengan asumsi file tersebut, misalnya `HelloWorld.java`, berada di direktori kerja saat ini, ketik perintah `javac` di diwah ini untuk mengompilasinya.

```batchfile
C:\Users\username> javac HelloWorld.java
```
